Banco de Dados Local

Configure um diretório com arquivos Parquet locais para evitar downloads repetidos.

Nenhum banco local configurado

Configurar Diretório

Aponte para um diretório contendo arquivos Parquet do geobr.

Python
import geobr
geobr.set_local_db("/caminho/para/seus/parquets/")
R
library(geobr)
set_local_db("/caminho/para/seus/parquets/")

Consultar & Limpar

Verifique o banco atual ou remova a configuração para voltar a usar a rede.

Python
# Consultar caminho atual
geobr.get_local_db()

# Remover configuração
geobr.clear_local_db()
R
# Consultar caminho atual
get_local_db()

# Remover configuração
clear_local_db()

Listar Datasets Locais

Lista todos os arquivos Parquet reconhecidos no diretório configurado.

📋 Convenção de nomes

municipalities_2022_simplified.parquet → simplified=True, year=2022
states_2020_original.parquet → simplified=False, year=2020
biomes_2019_simplified.parquet → Biomas 2019 simplificado
Python
# Retorna um DataFrame com: geography, year, simplified, path
df = geobr.list_local_datasets()
print(df)
R
# Retorna um data.frame com: geography, year, simplified, path
df <- list_local_datasets()
print(df)

Listar Datasets

Consulte todos os datasets disponíveis na API do geobr.

Python 
import geobr

# Retorna um DataFrame com todos os datasets disponíveis
df = geobr.list_geobr()
print(df)
R 
library(geobr)

# Retorna um tibble com todos os datasets disponíveis
df <- list_geobr()
print(df)
